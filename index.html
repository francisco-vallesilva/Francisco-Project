Abstract

Designing efficient aircraft wings requires evaluating thousands of airfoil shapes across many flight conditions. The standard tool, XFOIL — a panel-method physics solver — is accurate but slow: each simulation can take seconds to minutes, and a full design sweep demands thousands of runs. This project trains a ResNet-18 convolutional neural network as a surrogate model that predicts the lift coefficient (CL) of an airfoil directly from its 2D silhouette image, eliminating the solver entirely at inference time. The model is trained on real XFOIL simulations at a fixed Reynolds number of 1,000,000 and angle of attack of 5°, achieving R² = 0.907 on a held-out validation set — demonstrating that meaningful aerodynamic performance can be inferred from shape geometry alone, in milliseconds.

Dataset & Pipeline

From coordinates to labeled images

Every airfoil in the UIUC database starts as a plain text file of (x, y) surface coordinates. Each shape was run through XFOIL to obtain its lift coefficient at a fixed operating condition, then rendered as a 224×224 black-and-white silhouette — the format that ResNet-18 expects as input.

Model

ResNet-18 adapted for lift regression

Rather than training from scratch, the model uses transfer learning: ResNet-18's pretrained ImageNet weights give it a strong prior for detecting edges and shapes. All layers except the final convolutional block (layer4) are frozen — only layer4 and a new regression head are trained on the airfoil data. This constrains the hypothesis space and prevents overfitting on the 854-sample dataset.

Experiments

Hyperparameter search across 13 runs
